## Deutsch

ExternalDriveTester ist ein modernes, benutzerfreundliches Testprogramm fuer externe Datentraeger (wie USB-Sticks und externe Festplatten).

Funktionen:

- Kapazitaetspruefung (verifizierte Schreib-/Lese-Tests)
- Geschwindigkeitsmessung (Schreib-/Lesegeschwindigkeit)
- Integritaetstest (Datenkonsistenzpruefung)
- Backup/Restore ganzer Laufwerke als ZIP-Archiv

Konfiguration ueber externe JSON-Datei (z. B. unterstuetzte Sprachen, Theme, Testgroessen)

Fortschrittsanzeige und Ergebnisprotokollierung in der UI

## English

ExternalDriveTester is a modern, user-friendly testing tool for external storage devices (such as USB flash drives and external hard drives).

Features:

- Capacity check (verified write/read tests)
- Speed measurement (write/read throughput)
- Integrity test (data consistency check)
- Backup/restore of complete drives as ZIP archive

Configuration via an external JSON file (e.g. supported languages, theme, test sizes)

Progress display and result logging in the UI
